# SeleniumJavaTestNGDemo

Maven-based Selenium + TestNG demo scaffold.

Structure:
- `pom.xml` - Maven configuration
- `testng.xml` - TestNG suite
- `src/main/java/pages` - Page Object classes
- `src/main/java/utils` - Utilities (browser, config, excel, screenshots)
- `src/test/java/tests` - Test classes

To run tests:

```bash
mvn test
```
